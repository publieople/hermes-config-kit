# 支付宝 ↔ 余额宝 多账户资金流转

## 渠道约定

⚠️ **重要：不存在「支付宝余额」渠道。** 支付宝余额变动统一用 **「支付宝」**。

| 渠道 | 适用场景 |
|------|----------|
| **支付宝** | 支付宝余额的所有变动（回收款到账、转入余额宝、余额宝转回余额、提现到银行卡） |
| **余额宝** | 余额宝内部交易（自动转入、转出、收益） |
| 中国银行社保卡 | 提现目标卡 |
| 中国银行储蓄卡 | 常规储蓄卡扣款 |

## 完整链路（双账户视角，7 笔记录）

## 关键规则

1. **金额全正数**：方向和正负由大项区分（收入/流转入=流入，流转出/饮食=流出）
2. **标签必填**：所有资金流转记录加「资金流转」标签，提现加「提现」
3. **标题不含emoji**：emoji 通过 icon 属性设置
4. **渠道精确**：支付宝余额用「支付宝」，余额宝内部用「余额宝」
5. **父项关联**：相关记录通过父项串联，形成可追溯的链路
