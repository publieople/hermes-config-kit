# GB/T 7714-2015 参考文献著录规则 — 速查

## 文献类型标识
| 类型 | 标识 | 说明 |
|------|------|------|
| 普通图书 | [M] | monograph |
| 期刊文章 | [J] | journal |
| 学位论文 | [D] | dissertation |
| 论文集 | [C] | collection |
| 报告 | [R] | report |
| 专利 | [P] | patent |
| 电子资源 | [EB/OL] | electronic/online |
| 报纸 | [N] | newspaper |

## 格式模板

### 专著 [M]
```
作者. 书名[M]. 版本(第一版不注). 出版地: 出版社, 年份[: 页码].
```
例: `程俊英. 诗经译注[M]. 上海: 上海古籍出版社, 2012: 15-20.`

### 译著
```
作者. 书名[M]. 译者, 译. 出版地: 出版社, 年份.
```
例: `尼葛洛庞帝. 数字化生存[M]. 胡泳, 范海燕, 译. 海口: 海南出版社, 1996.`

### 期刊文章 [J]
```
作者. 题名[J]. 刊名, 年份, 卷(期): 页码.
```
例: `杨宗英. 电子图书馆的现实模型[J]. 中国图书馆学报, 1996(2): 24-29.`

### 学位论文 [D]
```
作者. 题名[D]. 出版地: 学校名称, 年份.
```

### 电子资源
```
作者. 题名[J/OL]. 刊名, 年份, 卷(期): 页码[引用日期]. 获取和访问路径.
```
例: `储大同. 恶性肿瘤个体化治疗靶向药物的临床表现[J/OL]. 中华肿瘤杂志, 2010, 32(10): 721-724[2014-06-25]. http://vip.calis.edu.cn/asp/Detail.asp.`

## 著录用符号
- `.` 用于各著录项目之间
- `,` 用于同一项目内的子项
- `:` 用于出版地与出版者之间、年与页码之间
- `//` 用于析出文献与源文献之间
- `()` 用于期号、版本等
- `[]` 用于文献类型标识、引用日期

## 注意事项
1. **作者:** 3人以内全部列出，用","分隔；超过3人列前3人加",等"
2. **出版地:** 指城市名（如"北京"），非省名
3. **版本:** 第一版不标注，第2版起标注
4. **页码:** 引用页码（非文章起止页码）用于引文标注
5. **外国作者:** 姓在前全大写，名用缩写（如 `DOWLER L`）
6. **出版年:** 公元纪年，报纸用 `YYYY-MM-DD` 格式

## 常见著录示例
```
[1] 扬之水. 诗经别裁[M]. 北京: 中华书局, 2012: 45-52.
[2] 余冠英. 诗经选[M]. 北京: 人民文学出版社, 1979: 28-30.
[3] 冯时. 《诗经》日食考[J]. 中国社会科学院大学学报, 2025, 45(7): 42-55.
[4] 谢炳军. 安大简《诗经》文本编纂的三个思想倾向[J]. 思想与文化, 2020(2): 21-34.
[5] 高亨. 诗经今注[M]. 上海: 上海古籍出版社, 1980.
[6] 潘富俊. 草木缘情：中国古典文学中的植物世界[M]. 2版. 北京: 商务印书馆, 2016.
[7] 周雨菲. 《诗经》桑意象的文化内涵探赜[J]. 国学, 2025, 13(2): 310-317.
```

## 引用标注（顺序编码制）
- 正文中用 `[1]` `[2]` 上标标注，按出现顺序编号
- 同一处引用多篇: `[5,70,83]` 或连续号 `[255-256]`
- 多次引用同一文献: 正文标注首次序号，方括号外注页码，如 `[2]260`
